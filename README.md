# Longest Path in a Grid Maze

## Project Description
This project solves the Longest Path Problem in a grid graph, which is known to be an NP-Hard problem (a general form of the Hamiltonian path problem). It uses a Backtracking (Depth-First Search) algorithm with pruning to find the longest continuous path of unvisited open cells in a maze.

## How to Compile
Ensure you have the Java Development Kit (JDK) installed. Open your terminal, navigate to the `src` directory, and run:
```bash
javac LongestPathMaze.java
